# ATM Console Application

## Description
This is a **mini console-based ATM project** created to demonstrate basic programming and banking operations.  
The program allows users to securely log in to their account and perform transactions such as deposits, withdrawals, balance inquiries, and PIN changes.  
It is designed to simulate a real ATM experience using a simple command-line interface.

## Features
- User authentication with account number and PIN  
- Deposit money into the account  
- Withdraw money with balance check  
- View transaction history  
- Check account balance  
- Change account PIN securely  
- Exit option to end session  

## How to Run
1. Open the project file (`.py`) in any Python IDE or console  
2. Run the file  
3. Enter a valid account number from the predefined users  
4. Enter the corresponding PIN  
5. Follow the menu options to perform transactions  

## Technologies Used
- Python  
- Console-based input/output  
- Lists and dictionaries for data management
